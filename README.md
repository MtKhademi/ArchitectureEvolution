# 🚀 دوره جامع معماری نرم‌افزار: مسیر تعالی از مبانی تا میکروسرویس

به این دوره آموزشی خوش آمدید! این مخزن یک سفر عملی و گام‌به‌گام برای تبدیل شدن از یک توسعه‌دهنده ماهر به یک **معمار نرم‌افزار حرفه‌ای** است. ما در طول ۲۰ گام، یک سیستم مدیریت کالا و سفارش‌گیری واقعی را با تکنولوژی **NET 8** می‌سازیم.

## 🎯 فلسفه آموزش
این دوره بر اساس یک اصل ساده طراحی شده: **"هر گام، یک مشکل واقعی را حل می‌کند."** ما مفاهیم را به ترتیبی یاد می‌گیریم که انگار یک پروژه واقعی از روز اول تا Production پیش می‌رود. خبری از آموزش جزیره‌ای و بی‌ربط نیست!

## 📊 پروژه: سیستم مدیریت کالا و سفارش‌گیری (OMS)
سیستم ما یک پلتفرم کامل با این قابلیت‌هاست:
- **User Management**: ثبت‌نام، ورود، پروفایل کاربری
- **Product Management**: تعریف کالا، دسته‌بندی، قیمت‌گذاری، موجودی
- **Order Management**: ایجاد سفارش، وضعیت‌های مختلف، تاریخچه
- **Role-based Access Control**: سطح دسترسی‌های پیشرفته
- **Reporting**: گزارش‌های فروش، موجودی، و تحلیل‌های حرفه‌ای

## 🗺️ مسیر یادگیری: گام ها تا معماری حرفه‌ای

| Stage | برنچ | عنوان | سوال اصلی این مرحله |
|-------|------|-------|-------------------|

## 🛠️ تکنولوژی‌های اصلی
- **زبان برنامه‌نویسی:** C# 12 (.NET 8)
- **فریم‌ورک وب:** ASP.NET Core Web API
- **پایگاه داده:** SQL Server, PostgreSQL
- **ORM:** Entity Framework Core 8
- **کش:** Redis, In-Memory Cache
- **صف پیام:** RabbitMQ (MassTransit)
- **جستجو:** Elasticsearch
- **Mediator:** MediatR
- **اعتبارسنجی:** FluentValidation
- **نگاشت اشیاء:** Mapster
- **کارهای پس‌زمینه:** Hangfire
- **احراز هویت:** JWT, Refresh Token
- **لاگینگ:** Serilog
- **مشاهده‌پذیری:** OpenTelemetry, Jaeger, Prometheus, Grafana
- **مستندسازی:** Swagger/Scalar
- **گزارش‌گیری:** ClosedXML, QuestPDF
- **تست:** xUnit, NSubstitute, FluentAssertions
- **Container:** Docker, Docker Compose
